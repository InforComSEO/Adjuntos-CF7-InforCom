Sistema avanzado de carga de archivos drag & drop para Contact Form 7.

== Description ==

**Adjuntos CF7 - InforCom** es un plugin premium desarrollado por InforCom Soluciones Web que extiende Contact Form 7 con un sistema completo de carga de archivos drag & drop.

= Características principales =

* Panel de configuración global y por formulario
* Vista previa en tiempo real
* Almacenamiento local y Google Drive
* Log de archivos subidos
* Sistema de errores 100% personalizable
* Seguridad avanzada con validación MIME
* Limpieza automática de archivos huérfanos
* Exportar e importar configuración
* Accesibilidad WCAG

== Installation ==

1. Sube la carpeta `adjuntos-cf7-inforcom` a `/wp-content/plugins/`
2. Activa el plugin desde el menú Plugins de WordPress
3. Ve a **Adjuntos CF7 > Configuración** para configurar el plugin
4. Agrega el tag `[adjuntos-cf7 nombre]` a tu formulario CF7
